🚀 БЫСТРЫЙ СТАРТ (Выберите один способ)
═══════════════════════════════════════════════════════════════════════════════

Способ 1: Windows (Самый простой)
   1. Дважды щелкните: start_dashboard.bat
   2. Дождитесь открытия терминала
   3. Откройте браузер: http://localhost:8000

Способ 2: Linux/Mac
   1. chmod +x start_dashboard.sh
   2. ./start_dashboard.sh
   3. Откройте браузер: http://localhost:8000

Способ 3: Вручную (любая ОС)
   1. Откройте терминал
   2. cd c:\bybit-trading-bot
   3. python run_api.py
   4. Откройте: http://localhost:8000

🔌 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

КОНФИГУРАЦИЯ:
   GET  /api/config              - Получить всю конфигурацию
   GET  /api/config/{section}    - Получить раздел
   POST /api/config/{key}        - Обновить значение
   POST /api/config/validate     - Валидировать
   POST /api/config/reset        - Сбросить на дефолты

ТОРГОВЛЯ:
   GET  /api/trading/history     - История сигналов
   GET  /api/trading/orders      - История ордеров
   GET  /api/trading/executions  - Исполнения
   GET  /api/trading/stats       - Статистика

АККАУНТ:
   GET  /api/account/balance     - Информация о балансе
   GET  /api/account/positions   - Открытые позиции
   GET  /api/account/status      - Статус аккаунта

LIVE:
   WS   /ws                      - WebSocket для обновлений

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
═══════════════════════════════════════════════════════════════════════════════

В: Какой порт используется?
О: По умолчанию 8000. Измените в run_api.py если нужно.

В: Как получить доступ с другого компьютера?
О: Используйте IP сервера: http://192.168.1.100:8000

В: Могу ли я интегрировать со своим ботом?
О: Да! API автоматически работает с вашим ботом.

В: Как развернуть в production?
О: Используйте Docker или развертите на облачный сервер.

В: Есть ли примеры использования API?
О: Да, см. DASHBOARD_SETUP.md и API_DASHBOARD.md

В: Как изменить конфигурацию?
О: Используйте Settings вкладку или API POST /api/config/{key}

В: Поддерживается ли мобильный доступ?
О: Да! Панель полностью отзывчива для мобильных устройств.
