✨ 4 ГЛАВНЫХ ВКЛАДКИ

   1️⃣  DASHBOARD (Обзор в реальном времени)
       • Статус бота (режим, символ, уровень риска)
       • Баланс аккаунта и нереализованный PnL
       • Количество открытых позиций
       • Статистика сигналов и последние сделки
       • Автоматическое обновление каждые 10 секунд

   2️⃣  SETTINGS (Управление конфигурацией)
       • Редактирование торгового символа и режима (Paper/Live)
       • Управление рисками (% позиции, макс плечо)
       • Stop Loss и Take Profit уровни
       • Мгновенное сохранение изменений
       • Опция сброса на дефолты

   3️⃣  HISTORY (История торговли)
       • Таблица сигналов с временем
       • Таблица ордеров со статусом
       • Таблица исполнений с ценами
       • Готово к пагинации
       • Сортируемые колонки

   4️⃣  ACCOUNT (Детали аккаунта)
       • Информация об аккаунте (ID, статус, тип)
       • Разбор баланса (всего, доступно, PnL)
       • Открытые позиции с ценами входа/выхода
       • Real-time отслеживание P&L

📋 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ API
════════════════════════════════════════════════════════════════════════════════

PYTHON ПРИМЕР:

import requests

# Получить баланс аккаунта
response = requests.get('http://localhost:8000/api/account/balance')
balance = response.json()
print(f"Баланс: {balance['total_balance']} USDT")

# Получить открытые позиции
response = requests.get('http://localhost:8000/api/account/positions')
positions = response.json()
for pos in positions:
    print(f"{pos['symbol']}: {pos['size']} @ {pos['entry_price']}")

# Обновить торговый символ
response = requests.post(
    'http://localhost:8000/api/config/trading.symbol',
    json={'value': 'ETHUSDT'}
)
print(response.json())


JAVASCRIPT ПРИМЕР:

// Получить конфигурацию
fetch('http://localhost:8000/api/config')
  .then(response => response.json())
  .then(config => console.log(config));

// Обновить процент риска
fetch('http://localhost:8000/api/config/risk_management.position_risk_percent', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({value: 2.5})
})
.then(response => response.json())
.then(data => console.log(data));

// WebSocket для real-time обновлений
const ws = new WebSocket('ws://localhost:8000/ws');
ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Real-time update:', data);
};
